
# Stock Market Simulator
#### Video Demo:  <https://youtu.be/ZJvViv5t6cI>
#### Description:
This is a simple command line which can simulate buying and selling stocks.
It saves all of your data in holdings.csv and report.csv
# If it doesn't work, try updating yfinance pip install --upgrade yfinance --no-cache-dir

## __Requirements__
Required libraries can be downloaded by
```pip install -r requirements.txt```


## __Usage__
For the first time it will show the user some tasks to choose from. User can do as many tasks as possible before running out of money. However, if money is not deposited already user will not be able to use the program. There is no limit on depositing money.\

**Note: "Q" is to be inputted if user wants to quit(). Pressing Control-C will result the same.**

Here is the first output which will guide the user.
```
What do you want to do? (Write the desired quoted task.)
    "Report": Check your report
    "Buy": Buy a stock
    "Sell": Sell a stock
    "Reset": Resets everything
    "Q": Quit the program
```
